Features

主な機能

日々の業務に必要な記録をすべてカバー

業務前の点呼

点呼方法・確認者・アルコール検知・健康状態・指示事項を法定項目に沿って記録。

業務後の点呼

帰着後の点呼内容を記録。業務前点呼と合わせて1日の記録を一元管理。

業務記録（日報）

開始・終了地点、走行距離、主な経過地点（到着・出発時刻）、待機・荷役記録まで詳細に管理。

ヒヤリハット・事故記録

発生概要・原因・備考を一画面で記録。安全運転の振り返りと再発防止に。

月次カレンダー管理

月単位で記録状況を一覧表示。入力済みの日をひと目で確認でき、漏れを防げます。

PDF出力・共有

記録をPDF化してメールやAirDropで共有。そのまま提出書類として使えます。

自動入力（定型入力）

よく使う内容を登録しておき、ワンタップで各記録ページに反映。毎日の入力を大幅に短縮。

クラウド自動バックアップ

データはクラウドに自動同期。機種変更後も同じアカウントでログインするだけで即復元。

顔認証ログイン

Face ID / Touch ID で素早くログイン。毎回パスワードを入力する手間がありません。

How it works

1日の使い方

出発前から帰着後まで、3ステップで完結

1

出発前に「業務前の点呼」を記録

点呼方法・確認者・アルコール検知結果・健康状態・指示事項を入力。自動入力を使えば数秒で完了。

2

業務中・帰着後に「業務記録」を入力

走行距離・経過地点・待機・荷役情報を記録。必要に応じてヒヤリハット・事故の記録も追加できます。

3

帰着後に「業務後の点呼」を記録

帰着後の点呼内容を入力して1日の記録が完結。必要な日の記録をカレンダーからPDF出力できます。

FAQ

よくある質問

どのOSに対応していますか？

iOS（iPhone）およびAndroid（スマートフォン・タブレット）に対応しています。

招待コードとは何ですか？

登録時に招待コードを入力すると、初回限定の無料トライアル期間を利用できます。招待コードがない方は CARGO1 を入力することで1か月間の無料トライアルをご利用いただけます。

PDFを出力する方法は？

カレンダー画面で日付を長押しするとPDF出力メニューが開きます。出力したいセクション（点呼・業務記録など）を選択して「出力する」をタップすると、メールやAirDropなどで共有できます。

自動入力機能はどう使いますか？

設定画面の「自動入力の設定」から定型内容を登録しておきます。各記録ページの「自動入力」ボタンをタップすると、登録した内容がワンタップで反映されます。

機種変更してもデータは引き継げますか？

はい。クラウドバックアップにより、新しいスマートフォンに同じアカウントでログインするだけでデータを復元できます。

位置情報やカメラにアクセスしますか？

いいえ。位置情報（GPS）とカメラ・写真データは一切取得・利用しません。

解約はいつでもできますか？

はい。App StoreまたはGoogle Playのサブスクリプション設定からいつでも解約できます。解約後も期間終了まではご利用いただけます。
